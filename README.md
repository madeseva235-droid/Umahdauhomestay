# Website Homestay Nyaman

Proyek ini adalah contoh website statis bertema homestay.

## Struktur file

- `index.html` - halaman utama.
- `styles.css` - gaya tampilan.
- `script.js` - interaksi formulir sederhana.

## Menjalankan

Buka `index.html` di browser, atau jalankan dengan server lokal seperti Live Server di VS Code.

## Kustomisasi

Ubah teks, gambar, dan harga sesuai homestay Anda. Tambahkan halaman baru jika ingin memperluas fitur seperti reservasi atau testimonial.
